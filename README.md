# online pong-game

## sobre

Essa **cópia do jogo PONG** foi feita sem engines ou frameworks com o objetivo de treinar desenvolvimento Web

Esse jogo pode ser jogado sozinho ou multiplayer local tanto no computador quanto em mobile
